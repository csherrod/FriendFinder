# FriendFinder
An app to help you find some new friends.

https://thawing-escarpment-98135.herokuapp.com/

### Code Languages Used

The is built using html, javascript, jquery, node, express and bootstrap. 

### What it does

After filling out the survey and clicking the submit button the app will return a friend match based on the difference between each one of the answers in the survey and compare them to other people's answers.
